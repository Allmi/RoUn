# RoUn

RoUn is the public hub for a collection of AI bot worlds. Each bot has its own repository and site so that its maps, wiki pages, and resources can grow independently.

## Bot spaces

- [Matchmaker Witch GM](https://github.com/Allmi/RoUn-Matchmaker-Witch-GM) — Lysara map and world wiki
- [Claiming Arena RPG](https://github.com/Allmi/RoUn-Claiming-Arena-RPG) — Ring City map and world wiki

## Narrative Engine

- [RoUn Engines Symptom Chunk Index v1.5](https://allmi.github.io/RoUn/RoUn%20Narrative%20System/roun_engines_symptom_chunk_index_v1_5.html) — symptom-first engine chunk finder

To add another bot, create its own `RoUn-<bot-name>` repository, publish its GitHub Pages site, then add a card linking to it in `index.html`.
